# Stable Bot AGENTS

## 목적

이 폴더는 유전 알고리즘이 아닌 안정 완주 봇을 개발하기 위한 작업 공간입니다.

목표는 한 번 높은 점수를 내는 것이 아니라, 반복해서 `score=400`을 안정적으로 달성하는 것입니다.

## 작업 규칙

- `stable_bot/`은 `ga_bot/`과 분리해서 관리합니다.
- 사용자가 명시적으로 요청하지 않는 한 GA 코드를 삭제하거나 대규모로 수정하지 않습니다.
- 설명 가능한 결정론적 계획 로직을 우선합니다.
- 새 전략을 구현하기 전에는 `DESIGN_NOTES.md`에 설계 이유를 먼저 기록합니다.
- 함수는 설명 가능한 단위로 나눕니다.
- 가능한 경우 기존 공통 helper인 `snake_core.py`를 재사용합니다.
- 화면 실행이 필요하면 기존 `snake_game.run_game()` 흐름과 연결하는 방식을 우선 검토합니다.

## 예상 파일

- `hamiltonian.py`: 보드 전체를 순회하는 fallback 경로
- `safety.py`: 충돌, 꼬리 도달 가능성, 고립 공간 검사
- `planner.py`: 최종 이동 방향 결정
- `runner.py`: stable bot 실행 또는 반복 평가 진입점

## 검증 규칙

성공 여부는 한 판으로 판단하지 않습니다.

```text
100판 반복 평가
score=400 성공률 80% 이상
```

위 기준을 통과해야 안정 봇으로 봅니다.
